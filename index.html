About Me

I'm a musician, a performer, a songwriter and a Web Developer. I have been programming for 6 months, starting with JavaScript, HTML and CSS. I learned basic web development using jQuery, styling with CSS flexbox, Bootstrap and SASS, and AJAX for asynchronous interaction with the API. For my first full-stack application, I created a meme-generator front-end using handlebars and vanilla JS, plus a database with a Ruby/Rails framework and PostgreSQL. With more practice and experience coding in JavaScript, I also developed a dog matching app entirely in JS, using a React.JS frontend + Express server with MongoDB.

Recent Projects

Bazaar Memes

Bazaar Memes is a full-stack web app with a front-end meme generator and back-end meme database. Users can sign up, generate memes and share them with users from around the world. Add comments and votes on other memes and sort by upvotes and number of comments.

Wagger

Wagger is a full-stack web app built with React.js and Express/Mongoose/MongoDB. Users can sign up, create a dog profile, and match with other dogs in your area. Look through all your matches and message them to arrange playdates. Update every aspect of your profile to make sure you're always putting your best paw forward.

Let's Chat

Let's Chat is a full-stack web app built with Socket.io, jQuery, and handlebars. The backend also integrates Socket.io with a Node.JS Express server. Users can sign up, and chat in real-time with other users from potentially anywhere in the world. Real-time updates occur when another user is typing and when the chatroom receives a new message. Users can also update and delete their own messages if they made a typo or otherwise bad message. Updated messages are also updated for other users in real-time using Socket.io

Tic Tac Toe

This application allows a user to sign up for an account and play multiple rounds of tic-tac-toe. Game data can be displayed for the user including total games completed, win/loss/tie ratio, and win percentage.
